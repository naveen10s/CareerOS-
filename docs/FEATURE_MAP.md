# Feature Map Spec - CareerOS

This spec details the feature matrices and functional parameters grouped across each module in **CareerOS**.

---

## 🛠️ 1. Dashboard Module

- **Scope**: Primary workspace cockpit showing candidate status parameters.
- **Key Features**:
  - **Metric Aggregator**: Stat cards demonstrating velocity.
  - **Agile Progress Indicator**: Circular metrics displaying upskilling points.
  - **Activity Log**: Event log showing transition details.
  - **System Command Bar Link**: Direct trigger buttons to launch the command portal.

---

## 📊 2. Analytics Telemetry Module

- **Scope**: Displays quantitative proof of analytical competency.
- **Key Features**:
  - **Database Query Inspection**: Displaying SQL code blocks showing cohort calculation logic.
  - **Telemetry Latency Funnels**: Interactive graphs representing drop-off points.
  - **KPI Definition Matrix**: Explaining how Business Analysis actions translate into Product metrics.

---

## 🗺️ 3. Journey Timeline Module

- **Scope**: Visual storytelling mapping BA to PM progression.
- **Key Features**:
  - **Vertical Scroll Timeline**: Chronological events using GSAP split text.
  - **Competency Mapping node**: Highlighting when SQL or process mapping skills were utilized.
  - **Filter controls**: View all milestones vs transition actions only.

---

## 📂 4. Projects Module

- **Scope**: Portfolio case studies proving problem solving.
- **Key Features**:
  - **Spotlight Case Cards**: Cards with tilt reactions.
  - **PM/BA Tag indicators**: Displaying exactly what skills were shown in each project.
  - **Wireframe carousel stubs**: Slide carousels showing prototypes.

---

## 📄 5. Resumé Module

- **Scope**: Career summaries formatted for recruiters.
- **Key Features**:
  - **Interactive PDF Print**: CSS rules to generate high-fidelity standard printable documents.
  - **Transferable Highlights**: Color highlights linking experience details back to skills.

---

## 📚 6. Learning & Upskilling Module

- **Scope**: Active learning engine.
- **Key Features**:
  - **Certifications Tracker**: CSPO, Agile certifications listing.
  - **Reading log**: Book summaries and key product takeaways.
  - **Upskill progress**: Linear bars representing path completions.

---

## ✍️ 7. Blog Module

- **Scope**: Industry posts showing product writing capabilities.
- **Key Features**:
  - **Markdown article parser**: Renders text documents.
  - **Category filters**: Sort articles by Agile, User Discovery, SQL.

---

## 📞 8. Contact & Booking Module

- **Scope**: Calendar scheduler.
- **Key Features**:
  - **Inline booking widget**: Select interview days.
  - **Social redirect shortcuts**: Hover indicator buttons leading to GitHub and LinkedIn.
